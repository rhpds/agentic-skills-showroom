= Agent Skills Showroom

A hands-on course teaching how to create, use, and distribute Agent Skills for Claude Code, built with https://antora.org[Antora] and the https://github.com/rhpds/showroom_template_nookbag[Red Hat Demo Platform Showroom] framework.

== Course Modules

1. **Introduction to Agent Skills** — history, installing skills, implicit vs explicit triggering
2. **Creating Your First Skill** — skill anatomy, frontmatter, progressive disclosure, building `talk-outliner`
3. **More Advanced Skills** — using `skill-creator` from the Anthropic marketplace to build `abstract-writer`
4. **Marketplaces and Namespacing** — bundling skills into a plugin, creating a git repo marketplace
5. **The Full Monty** — layered skills: superpowers harness + beads + third-party marketplace
6. **Bonus: Programmatic Skills** — skills in Deep Agents, skills calling skills
